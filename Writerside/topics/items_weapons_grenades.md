# Grenades

## dictionary
This json contains a dictionary of key: value pairs, with the keys representing the grenade itemIds, with the values
being a dictionary containing weapon attributes.

```json
{
	"1": { // <- Grenade Item ID
		"name": "Grenade Name",
		"description": "Grenade Description",
		"damage": 400,
		"penetration": 4,
		"outer_radius": 7,
		"fuse_time": 3.5
	}
}
```
