# Fire Modes

## dictionary
This json contains a dictionary of key: value pairs, representing the arbitrarily assigned Weapon Fire Mode IDs, with 
the values being string representations of the weapon fire mode names.

```json
{
  "0": "Semi Automatic",
  "1": "Burst",
  "2": "Automatic",
  "3": "Pump Action",
  "4": "Double Action"
}
```
